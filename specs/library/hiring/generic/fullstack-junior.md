# What We're Looking For

I'm looking for someone who's built something real from scratch — even if it's messy, even if it's small. A personal project that actually works is worth more than a dozen tutorial follow-alongs. I want to see curiosity and ownership, not perfection.

The best junior engineers I've hired had strong fundamentals and could explain their thinking clearly. They might not know every React hook or database optimization trick, but they understand why their code works and can debug when it doesn't. Most importantly, they're excited to learn the things they don't know yet.

## Must-Haves

- Built and deployed a full-stack application (frontend + backend + database)
- Comfortable with Git beyond just `git add` and `git commit`
- Can debug their own code using browser dev tools or server logs
- Has written tests, even basic ones, and understands why they matter
- Can explain the difference between client-side and server-side rendering

## Strong Signals

- Personal projects on GitHub with real commit history (not just final uploads)
- Has used both SQL and NoSQL databases and has opinions about when to use each
- Experience with any cloud platform (AWS, Vercel, Heroku, etc.)
- Has broken something in production and learned from it
- Contributes to tech communities (blog posts, Stack Overflow, Discord help)

## Anti-Patterns

- Only has bootcamp or tutorial projects with no personal exploration
- Can't explain their code choices beyond "that's what the tutorial said"
- Dismisses testing as "too complicated" or "not necessary for small projects"
- Wants to use the newest framework for everything without understanding the tradeoffs
- Can't work without heavy hand-holding or step-by-step instructions

## Interview Focus

- Walk through their best personal project: what they built, what went wrong, what they'd improve
- How they approach debugging when something breaks
- What they do when they encounter a technology they've never used before
- Their learning process: how they stay current and pick up new skills
